CMPS 3500 MiniScheme Hidden Demo Cases

This package contains 3 hidden demo cases intended for live presentations.
Keep these files separate from the public package and separate from the
larger hidden grading set.

Files:
- hidden_demo_01.scm  Lexical scope / shadowing
- hidden_demo_02.scm  Higher-order function application
- hidden_demo_03.scm  cond add-on behavior

Suggested usage:
- Choose one file per group during the live demo.
- Run it through the required command-driven runner.
- Do not distribute these files to students in advance.
